# Windows 11 Portfolio

A creative personal portfolio website inspired by the **Windows 11** desktop experience.  
This project simulates a fully interactive Windows 11-like desktop environment using Next.js, Tailwind CSS, and React.

- Beautiful Windows 11-themed desktop with custom wallpaper

  - System tray (Wi-Fi, volume, battery icons)
  - Live clock showing real-time based on user's timezone and locale

- Interactive Resume window:
